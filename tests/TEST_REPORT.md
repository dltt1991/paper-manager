## 4. 发现的Bug列表

### 4.1 Bug详情

#### BUG-001: 论文删除时文件系统延迟

| 项目 | 内容 |
|-----|------|
| **ID** | BUG-001 |
| **标题** | 论文删除时文件系统删除有延迟 |
| **严重程度** | 中 |
| **优先级** | P2 |
| **状态** | 待修复 |
| **发现时间** | 2026-02-26 |
| **所属模块** | 论文管理 |
| **测试用例** | TC-DEL-001 |

**描述**: 在删除论文时，数据库记录被删除，但文件系统偶尔会出现删除延迟。在高并发删除场景下可能导致文件残留。

**复现步骤**:
1. 创建多篇论文
2. 快速连续删除多篇论文
3. 检查上传目录，可能发现残留文件

**预期结果**: 文件应立即删除

**实际结果**: 文件删除有延迟，偶尔残留

**建议修复**: 使用文件删除队列或异步任务确保文件删除

---

#### BUG-002: PDF翻页最后一页判断问题

| 项目 | 内容 |
|-----|------|
| **ID** | BUG-002 |
| **标题** | PDF最后一页时下一页按钮未禁用 |
| **严重程度** | 低 |
| **优先级** | P3 |
| **状态** | 待修复 |
| **发现时间** | 2026-02-26 |
| **所属模块** | PDF预览 |
| **测试用例** | TC-PDF-UI-003 |

**描述**: 当PDF翻到最后一页时，下一页按钮应该被禁用，但目前仍可点击。

**复现步骤**:
1. 打开一个多页PDF
2. 翻到最后一页
3. 观察下一页按钮状态

**预期结果**: 下一页按钮禁用

**实际结果**: 按钮仍可点击，点击后无反应

**建议修复**: 在pageNumber >= numPages时正确禁用按钮

---

#### BUG-003: 颜色选择器响应问题

| 项目 | 内容 |
|-----|------|
| **ID** | BUG-003 |
| **标题** | 颜色选择器快速点击偶尔不响应 |
| **严重程度** | 低 |
| **优先级** | P3 |
| **状态** | 待修复 |
| **发现时间** | 2026-02-26 |
| **所属模块** | 批注功能 |
| **测试用例** | TC-ANN-UI-001 |

**描述**: 在批注工具中快速连续点击颜色选择器时，偶尔会出现不响应的情况。

**复现步骤**:
1. 打开PDF预览
2. 选择批注工具
3. 快速连续点击不同颜色

**预期结果**: 颜色正常切换

**实际结果**: 偶尔出现点击无响应

**建议修复**: 添加防抖或检查颜色选择器状态管理
